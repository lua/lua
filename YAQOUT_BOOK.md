# كتاب: احتراف البرمجة بلغة ياقوت 💎
## الدليل الشامل لتعليم البرمجة باللغة العربية
**تأليف: إسلام النشار (ستوديو النشار)**
**الإصدار: 1.0 (2026)**

---

## 📋 مقدمة الكتاب
مرحباً بك في عالم البرمجة العربية. لغة **ياقوت** ليست مجرد ترجمة للغة Lua، بل هي بيئة متكاملة تهدف لإزالة حاجز اللغة أمام العقول المبدعة في الوطن العربي. هذا الكتاب صُمم ليكون مرجعاً تعليمياً يبدأ معك من الصفر حتى بناء أنظمة برمجية معقدة.

---

## 📖 الفهرس
1.  **الفصل الأول**: إعداد بيئة العمل والترحيب الأول.
2.  **الفصل الثاني**: لبنات البناء (المتغيرات وأنواع البيانات).
3.  **الفصل الثالث**: المنطق واتخاذ القرار (الجمل الشرطية).
4.  **الفصل الرابع**: التكرار والقوة (الحلقات التكرارية).
5.  **الفصل الخامس**: هندسة الكود (الدوال).
6.  **الفصل السادس**: هيكلة البيانات (الجداول والخرائط).
7.  **الفصل السابع**: المكتبات القياسية (التعامل مع النظام والرياضيات).
8.  **الفصل الثامن**: البرمجة المتقدمة (الروتينات المساعدة والجداول الوصفية).

---

## 🛠️ الفصل الأول: الترحيب الأول
لغة ياقوت لغة نصية سهلة التعلم. لنبدأ بكتابة كودك الأول:

```lua
اطبع("مرحباً بك في لغة ياقوت")
```
اسم الملف: `البداية.yq`
التشغيل: `yaqout.exe البداية.yq`

---

## 📦 الفصل الثاني: لبنات البناء (المتغيرات)
المتغيرات هي مخازن للمعلومات.

### أنواع البيانات:
1.  **رقم**: (مثل `10` أو `3.14`).
2.  **نص**: (مثل `"ياقوت"`).
3.  **منطقي**: (`صح` أو `خطأ`).
4.  **عدم**: (`عدم` تعني لا شيء).
5.  **جدول**: (هيكل البيانات الوحيد والقوي).

```lua
محلي العمر = 25
محلي الاسم = "إسلام"
محلي هل_محب_للبرمجة = صح
```

---

## 🚦 الفصل الثالث: اتخاذ القرار
البرمجة هي القدرة على اختيار المسار.

```lua
محلي درجة_الحرارة = 30

إذا درجة_الحرارة > 40 إذن
    اطبع("الجو حار جداً")
وإلا_إذا درجة_الحرارة > 20 إذن
    اطبع("الجو معتدل")
وإلا
    اطبع("الجو بارد")
نهاية
```

---

## 🔄 الفصل الرابع: التكرار (Loops)
لتنفيذ كود معين عدة مرات.

### حلقة "لكل":
```lua
اطبع("العد التنازلي:")
لكل ر = 5, 1, -1 افعل
    اطبع(ر)
نهاية
اطبع("انطلاق! 🚀")
```

---

## ⚙️ الفصل الخامس: هندسة الكود (الدوال)
الدالة هي كود يمكنك إعادة استخدامه.

```lua
دالة حساب_المساحة(الطول, العرض)
    ارجع الطول * العرض
نهاية

محلي مساحة_الغرفة = حساب_المساحة(5, 4)
اطبع("المساحة تساوي: " .. مساحة_الغرفة)
```

---

## 📊 الفصل السادس: الجداول (Tables)
الجدول في ياقوت هو مصفوفة، قاموس، وكائن في آن واحد.

```lua
محلي طالب = {
    اسم = "أحمد",
    عمر = 20,
    درجات = {90, 85, 95}
}

اطبع(طالب.اسم)
جدول.إدراج(طالب.درجات, 88)
```

---

## 📚 الفصل السابع: المكتبات المعربة
ياقوت تأتي بمكتبات قوية جاهزة للاستخدام باللغة العربية:

*   **نصوص.عكس(ن)**: لقلب النص.
*   **رياضيات.جذر(س)**: لحساب الجذر.
*   **نظام.تاريخ()**: لجلب التاريخ.

مثال شامل:
```lua
محلي تاريخ_اليوم = نظام.تاريخ("%Y-%m-%d")
اطبع("تاريخ اليوم هو: " .. تاريخ_اليوم)
```

---

## 🚀 الفصل الثامن: البرمجة المتقدمة
تسمح لك ياقوت بتغيير سلوك اللغة عبر **الجداول الوصفية (Metatables)**.

```lua
محلي جدول_محمي = {}
تعيين_الجدول_الوصفي(جدول_محمي, {
    __index = دالة(ط, مفتاح)
        ارجع "المفتاح [" .. مفتاح .. "] غير موجود!"
    نهاية
})

اطبع(جدول_محمي.بيانات) -- سيطبع رسالة الحماية
```
